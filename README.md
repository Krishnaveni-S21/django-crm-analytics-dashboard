# FarmerCRM Data Analytics Dashboard

## Overview
FarmerCRM is a role-based CRM platform built for agricultural outreach and lead management. The system supports lead tracking, follow-up monitoring, automated lead assignment, and analytics dashboards for conversion and field performance.

## Features
- Lead pipeline tracking and monitoring  
- Follow-up status dashboard  
- Role-based CRM access  
- Bulk data upload functionality  
- Automated lead assignment by region and crop  
- SQL-based analytics for:
  - Lead volume trends  
  - Agent workload analysis  
  - Conversion tracking  
- Performance dashboards for field agents and managers

## Tech Stack
- Python  
- Django  
- PostgreSQL  
- SQL  
- HTML/CSS (if used)  

## Key Analytics Metrics
- Lead conversion rates  
- Agent productivity metrics  
- Follow-up completion rates  
- Regional lead distribution trends

## Project Purpose
This project demonstrates:
- Backend development with Django  
- SQL-driven data analysis  
- Dashboard reporting and business metrics  
- CRM workflow automation
